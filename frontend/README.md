# NSE Admin Panel - React Frontend

Modern React + Vite admin panel for NSE FII/DII Data Collector.

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:3000`

Make sure the Flask backend is running on `http://localhost:5000`

## Build

Build for production:

```bash
npm run build
```
